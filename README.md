# Bank Management System (OOP in C++)

A simple **Object-Oriented Programming (OOP) project in C++** that simulates basic banking operations.  
Perfect for learning **classes, objects, encapsulation, and methods** in C++.

---

## Features

- Create a new bank account
- Deposit money into an account
- Withdraw money from an account
- Check account balance
- Menu-driven interface for easy use

---

## Technologies

- **Language:** C++
- **Concepts:** Object-Oriented Programming (OOP)
  - Classes & Objects
  - Encapsulation
  - Methods
- **Data Structures:** `vector` (for storing multiple accounts)
